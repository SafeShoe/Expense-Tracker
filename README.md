# Expense-Tracker
Personal Expense Tracker 

# Overview 
A full-stack web application that allows users to record, view, update, and delete personal expense records. 


# Features 
User registration, Add, view, edit, delete expenses and data storage  


# Tech
HTML
CSS
JS


# Project Status 
Milestone 01 – Planning and Design
Milestone 02 - Front end, back end and design
Milestone 03 - Core functionality is complete, with final testing and documentation in progress



###
### Execution Instructions 
- Clone the repository from GitHub
- Open the project in VS Code
# Backend Setup
- Navigate to the Server folder
- Run npm install to install dependencies
- Start the server using npm start
- Backend will run on http://localhost:3001
# Database Setup
- Install and run MySQL locally
- Create a database 
- Create required tables using the provided SQL
- Ensure database connection details in the backend match your local setup
# Frontend Setup
- Open the Client folder 
- Run the frontend using Live Server 
- Register a new account
- Log in with your credentials
- Add, view, and manage expenses
# Configuration Notes
- Node.js must be installed
- MySQL must be installed and running
- Backend must be running before using the frontend
- Application uses localStorage to manage user sessions
Default ports:
Frontend: http://127.0.0.1:5500
Backend: http://localhost:3001
